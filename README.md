# Cryptex Frontend - Cryptocurrency Trading Platform UI

Cryptex Frontend is a modern and responsive **Cryptocurrency Trading Platform** built with **React.js**. It provides an interactive user interface for crypto trading, wallet management, transaction history, chatbot support, crypto news updates, and secure payment integration.

---

## 🚀 Project Overview

Cryptex Frontend delivers a seamless and user-friendly experience for users to buy/sell cryptocurrencies, manage wallets, transfer funds, view portfolios, access the latest crypto news, and interact with an AI chatbot.

The project is designed with a clean UI/UX and follows modern frontend development practices.

---

## 🛠️ Technology Stack

- **Framework:** React.js  
- **Build Tool:** Vite  
- **State Management:** Redux Toolkit  
- **Routing:** React Router DOM  
- **Styling:** Tailwind CSS / CSS  
- **UI Components:** shadcn/ui  
- **HTTP Client:** Axios  
- **Payment Integration:** Razorpay / Stripe  
- **Icons:** React Icons / Lucide React  
- **Notifications:** React Toastify / Sonner

## ⚙️ Getting Started

### Prerequisites

Make sure you have installed:

- Node.js  
- npm / yarn  

---

## 🔧 Installation & Setup

### Clone the Repository

    git clone https://github.com/yourusername/cryptex-frontend.git
    cd cryptex-frontend

### Install Dependencies

    npm install

### Run the Application

    npm run dev

The frontend will start on:

    http://localhost:5173

---

## 📬 API Connection

Connect frontend with backend by configuring API base URL.

Example:

    VITE_API_URL=http://localhost:5454
